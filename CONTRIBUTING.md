# Contributing to Glassdoor Job Scraper

Thank you for your interest in contributing to the Glassdoor Job Scraper project. By contributing to this project, you can help improve and enhance its functionality. Please take a moment to read this guide to understand how you can contribute effectively.

## Ways to Contribute

You can contribute to the project in the following ways:

1. Reporting Issues:
   - If you encounter any bugs, issues, or unexpected behavior while using the application, please open a GitHub issue. Be sure to provide detailed information about the problem, including steps to reproduce it and the environment you are using.

2. Submitting Enhancements:
   - You can submit enhancements or new features by opening a GitHub issue and describing the proposed changes. If you plan to work on the enhancement yourself, please indicate that in the issue.

3. Code Contributions:
   - If you want to contribute code changes, fork the repository and create a pull request (PR) with your proposed changes. Ensure that your code follows the project's coding standards and conventions.

## Getting Started

To contribute code, follow these steps:

1. Fork the repository to your GitHub account.

2. Clone the forked repository to your local machine:
   ```
   git clone https://github.com/omercngoktas/glassdoor-job-scraper-multiprocessing.git
   ```

3. Create a new branch for your changes:
   ```
   git checkout -b feature/your-feature-name
   ```

4. Make your code changes and ensure they meet the project's standards.

5. Commit your changes with a clear and descriptive commit message:
   ```
   git commit -m "Add feature: your feature description"
   ```

6. Push your changes to your forked repository:
   ```
   git push origin feature/your-feature-name
   ```

7. Create a pull request (PR) to the main project repository.

## Code of Conduct

Please note that this project follows the [Code of Conduct](CODE_OF_CONDUCT.md). By participating, you are expected to uphold this code.

Thank you for your contributions to the Glassdoor Job Scraper project!
